COGNOSCERE BookSummarizer

Panduan Setup & Testing di macOS

1
Pastikan Python Terinstall

BookSummarizer membutuhkan Python 3.8+. Cek apakah sudah terinstall — buka Terminal (Cmd+Space, ketik "Terminal"), lalu ketik:

python3 --version

Kalau muncul angka versi (misalnya Python 3.12.4), berarti sudah siap.

Belum punya Python?
Homebrew (disarankan): brew install python3
Download langsung: python.org/downloads/macos
2
Jalankan Aplikasi TANPA COMMAND LINE

Buka folder BookSummarizer di Finder, lalu:

🍎
start.command
Klik dua kali file ini di Finder
Pertama kali klik: macOS mungkin menampilkan peringatan keamanan.
Solusi: Klik kanan file → Open → klik Open di dialog konfirmasi.
Setelah itu, klik dua kali biasa akan selalu berfungsi.
Yang terjadi otomatis saat diklik:
1. Cek Python → Install dependencies yang belum ada
2. Buat file .env kalau belum ada
3. Jalankan web server Flask
4. Buka Safari/Chrome otomatis ke http://localhost:5000
3
Setup di Browser

A. Masukkan API Key Kimi

Buka halaman Settings (ikon gear di sidebar kiri), lalu paste API key kamu.

Belum punya? Daftar gratis di kimi.com/code/console → copy API key.

B. Pilih Folder Ebook

Masih di halaman Settings, ketik path folder yang berisi file-file PDF ebook kamu. Aplikasi akan otomatis scan semua PDF.

4
Mulai Proses Rangkuman

Library: Semua ebook PDF muncul sebagai kartu. Centang buku yang mau dirangkum, atau klik "Process All Pending" untuk proses semua.

Queue: Pantau progress real-time — progress bar, log aktivitas, estimasi waktu. Tombol Cancel tersedia kalau mau berhenti.

Estimasi: Sekitar 5–10 menit per buku. Kimi K2.5 butuh waktu karena menghasilkan review 8.000–15.000+ kata.

Setelah selesai:
Klik buku di Library → lihat rangkuman langsung di browser → download file DOCX.
Semua file tersimpan di folder output/ di dalam folder BookSummarizer.
5
Menghentikan & Melanjutkan

Menghentikan: Tutup jendela Terminal yang terbuka, atau tekan Ctrl+C di Terminal.

Melanjutkan: Klik dua kali start.command lagi. Progress tersimpan otomatis — aplikasi akan melanjutkan dari buku terakhir yang belum selesai.

FAQ

Apakah butuh koneksi internet?
Ya, koneksi internet diperlukan untuk mengirim teks ke API Kimi. Tapi UI aplikasinya sendiri berjalan lokal (localhost).
Port 5000 sudah dipakai, bagaimana?
Aplikasi otomatis mencari port alternatif (5001, 5002, dst). Tidak perlu konfigurasi manual.
File .env aman?
Ya, file .env berisi API key dan sudah masuk di .gitignore — tidak akan ter-push ke GitHub.
"start.command" tidak bisa dibuka?
Klik kanan → Open → klik Open di dialog. Atau buka Terminal dan jalankan: chmod +x start.command lalu klik dua kali lagi.
Bagaimana update ke versi terbaru?
Buka Terminal, masuk ke folder BookSummarizer, lalu jalankan git pull.
