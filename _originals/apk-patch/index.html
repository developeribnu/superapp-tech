Guide & Referensi

Command Reference

uber-apk-signer

Tool untuk signing APK dengan mudah:

java -jar uber-apk-signer.jar --apks myapp.apk \ --ks keystore.p12 \ --ks-pass yourpassword \ --ks-key-alias keyalias \ --key-pass keypassword

jarsigner (Built-in Java)

Tool bawaan Java untuk signing:

jarsigner -verbose -sigalg SHA256withRSA \ -digestalg SHA-256 \ -keystore keystore.jks \ -storepass yourpassword \ app-unsigned.apk alias_name

zipalign (Android SDK)

Optimize APK sebelum release:

zipalign -v 4 app-unsigned.apk app-aligned.apk

apksigner (Android SDK)

Verifikasi signature APK:

apksigner verify --verbose app.apk

Best Practices

🔐 Keystore Security

Simpan keystore di tempat aman, gunakan password yang kuat, dan jangan bagikan keystore dengan siapa pun. Keystore adalah kunci untuk update aplikasi Anda.

📱 Compatibility

Gunakan V1 dan V2 signing untuk kompatibilitas maksimal. V1 untuk Android 6.0 ke bawah, V2 untuk Android 7.0 ke atas.

✅ Verification

Selalu verifikasi APK sebelum release ke production. Pastikan signature dan manifest valid menggunakan tools yang tersedia.

🔄 Version Management

Kelola version code dan name dengan hati-hati. Version code harus increment untuk setiap update, version name untuk user readability.
