Common Network Attacks

DDoS Attack (Distributed Denial of Service)

Penyerang mengirim traffic berlebihan untuk membuat service tidak tersedia

Mitigation: Rate limiting, CDN, firewall rules

Man-in-the-Middle (MITM)

Penyerang intercept komunikasi untuk mencuri atau memodifikasi data

Mitigation: Encryption, certificate pinning, HSTS

Packet Sniffing

Capture unencrypted network traffic untuk ekstrak sensitive data

Mitigation: Use HTTPS, VPN, encrypt data in transit

ARP Spoofing

Manipulate ARP table untuk redirect traffic ke attacker machine

Mitigation: Dynamic ARP inspection, static ARP entries

OWASP Top 10 Security Checklist

OWASP Top 10 2024

#1: Broken Access Control

CRITICAL

Pengguna dapat mengakses resources yang seharusnya terbatas

Remediation: Implement proper authorization, use role-based access control (RBAC)

#2: Cryptographic Failures

CRITICAL

Data tidak diencrypt atau menggunakan encryption yang lemah

Remediation: Use strong encryption (AES-256), TLS 1.3, secure password hashing

#3: Injection

CRITICAL

SQL Injection, Command Injection, LDAP Injection attacks

Remediation: Use parameterized queries, input validation, ORM frameworks

#4: Insecure Design

HIGH

Missing security controls dalam design aplikasi

Remediation: Threat modeling, secure SDLC, security requirements

#5: Security Misconfiguration

HIGH

Default credentials, unnecessary services, verbose error messages

Remediation: Hardening guides, security baselines, configuration management

#6: Vulnerable & Outdated Components

HIGH

Using libraries dengan known vulnerabilities

Remediation: Dependency scanning, regular updates, SCA tools

#7: Authentication & Session Management

HIGH

Weak password policies, session hijacking, exposed credentials

Remediation: MFA, secure session management, password hashing

#8: Software & Data Integrity Failures

MEDIUM

Unsafe deserialization, unsigned updates

Remediation: Digital signatures, code signing, secure CI/CD

#9: Logging & Monitoring Failures

MEDIUM

Insufficient logging, tidak ada alerting untuk suspicious activities

Remediation: Comprehensive logging, SIEM solutions, alerting

#10: Server-Side Request Forgery (SSRF)

MEDIUM

Server membuat request ke attacker-controlled URLs

Remediation: Input validation, whitelist URLs, network segmentation

Challenge: Simple Cipher

Encrypted text: "KHOOR ZRUOG"

Hint: Caesar cipher dengan shift value tertentu

# Solution Approach:
# 1. Test semua kemungkinan shift (0-25)
# 2. Lihat mana yang menghasilkan meaningful text
# 3. Ekstrak flag dari plaintext

shift = 3  # Correct shift
plaintext = ""
for char in encrypted:
    if char.isalpha():
        plaintext += chr((ord(char) - ord('A') - shift) % 26 + ord('A'))
    else:
        plaintext += char

# Result: "HELLO WORLD"
