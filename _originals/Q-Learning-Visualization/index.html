Q-Learning Algorithm

Bellman Equation

Q(s, a) = r + γ max Q(s', a')

Nilai Q untuk state-action = reward + discounted max future value

Q(s, a)

Quality value untuk state-action pair

r

Immediate reward dari action

γ (gamma)

Discount factor (0-1), bobot future rewards

s', a'

Next state dan best action dari state tersebut

Update Rule

Q(s, a) ← Q(s, a) + α[r + γ max Q(s', a') - Q(s, a)]

α (alpha)

Learning rate, kontrol update speed

Typical: 0.1 - 0.5

Temporal Difference

r + γ max Q(s', a') - Q(s, a)

Prediction error

ε-Greedy

Dengan prob ε: random action, else: best action

Exploration vs Exploitation

Pseudocode

Initialize Q(s, a) = 0 for all s, a For episode = 1 to max_episodes: Initialize state s For step = 1 to max_steps: With probability ε: a = random action Else: a = argmax Q(s, a) Execute a, observe r, s' Q(s, a) = Q(s, a) + α[r + γ max Q(s', a') - Q(s, a)] s = s' if s is terminal: break Return optimal policy from Q-table

Hyperparameter Impact

Learning Rate (α)

  • α = 0.01 - Belajar lambat, stabil
  • α = 0.1 - Balance (recommended)
  • α = 0.9 - Belajar cepat, unstable

Discount Factor (γ)

  • γ = 0.1 - Myopic, fokus immediate reward
  • γ = 0.9 - Farsighted (recommended)
  • γ = 0.99 - Very long-term planning

Initial ε (Exploration)

  • ε = 0.1 - Sedikit eksplorasi
  • ε = 0.5 - Balanced exploration
  • ε = 0.9 - Banyak eksplorasi

Decay Rate (ε decay)

  • decay = 0.99 - Slow decay
  • decay = 0.995 - Medium decay
  • decay = 0.9 - Fast decay
