📚 Fondasi Prompt Engineering

Struktur Dasar Prompt

[ROLE]: Kamu adalah seorang {ahli}
[CONTEXT]: Kami sedang mengerjakan {proyek}
[TASK]: Buat {output}
[CONSTRAINTS]: Format {spesifik}
[EXAMPLES]: {contoh input/output}
[TONE]: Menggunakan gaya {tone}

✅ Prompt Baik

"Kamu adalah seorang data scientist expert. Analisis dataset e-commerce ini dan identifikasi 3 tren penjualan utama. Format dalam JSON dengan penjelasan singkat."

❌ Prompt Buruk

"Analisis data ini"

🎯 Teknik Dasar

  • • Few-shot Learning: Berikan 2-3 contoh untuk membantu AI memahami pola
  • • Specificity: Semakin spesifik, semakin baik hasil yang didapat
  • • Clarity: Hindari ambiguitas dan gunakan bahasa yang jelas
  • • Context Limit: Jangan berikan terlalu banyak konteks yang tidak relevan

🚀 Teknik Lanjut

🔗 Chain of Thought

Minta AI untuk menjelaskan pemikiran langkah demi langkah sebelum memberikan jawaban akhir.

"Mari kita pikirkan ini langkah demi langkah:\n1. Identifikasi masalah\n2. Analisis penyebabnya\n3. Tentukan solusi\n4. Berikan rekomendasi akhir"

🤔 Few-shot Prompting

Berikan beberapa contoh untuk memandu AI dalam menghasilkan output yang konsisten.

"Contoh 1: Input -> Output\nContoh 2: Input -> Output\nSekarang: Input -> ?"

🎭 Role-Based Prompting

Assign peran spesifik untuk meningkatkan kualitas dan konsistensi response.

"Kamu adalah seorang {expert/profesional}.\nTugas: {spesifik task}\nOutput format: {format yang diinginkan}"

🔄 Iterative Refinement

Gunakan feedback dari respons sebelumnya untuk menyempurnakan prompt berikutnya.

"Respons Anda terlalu formal. Coba lagi dengan gaya lebih casual.\nTambahkan juga contoh konkret."

💬 ChatGPT Mastery Tips

💡 Tip 1: Gunakan Persona

"Kamu adalah {role}. Respons dengan style {gaya}. Gunakan bahasa {bahasa}."

Hasilnya akan lebih konsisten dan sesuai dengan kebutuhan Anda.

🎯 Tip 2: Buat Koneksi Konteks

"Ingat bahwa saya {konteks penting}. Harus mempertimbangkan {faktor spesifik}."

Membantu ChatGPT memahami konteks unik situasi Anda.

🔄 Tip 3: Iterasi Berkelanjutan

"Buat lebih {atribut}. Kurangi {elemen}. Fokus pada {aspek}."

Refinement adalah kunci untuk hasil optimal.

📋 Tip 4: Format Output

"Format hasilnya sebagai: {JSON/Table/Markdown/List}. Struktur dengan: {cara spesifik}."

Spesifikasi format membuat output lebih mudah diproses.

🎨 Tip 5: Constraint Creativity

"Ide harus {kriteria spesifik}. Hindari {hal yang tidak diinginkan}. Fokus pada {area}."

Constraint yang tepat menghasilkan kreativitas yang lebih terarah.

🎨 Midjourney Prompt Mastery

📐 Struktur Ideal Midjourney Prompt

/imagine prompt: [Subjek] [Deskripsi Detail] [Style/Medium] [Lighting] [Mood] --ar [aspect ratio] --niji

✅ Prompt Baik

"cyberpunk city at night, neon lights, futuristic skyscrapers, flying cars, photorealistic, cinematic lighting, volumetric fog --ar 16:9"

❌ Prompt Lemah

"futuristic city"

🎯 Parameter Penting

  • --ar 16:9 (aspect ratio) - landscape
  • --ar 1:1 - square
  • --ar 9:16 - portrait
  • --niji - anime/illustration style
  • --style raw - less Midjourney style
  • --iw 2 - image weight (relative importance)
  • --q 2 - quality (0.25, 0.5, 1, 2)

🎨 Style Keywords

Tambahkan untuk hasil lebih spesifik:

• Photorealistic
• Oil painting
• Watercolor
• Digital art
• 3D render
• Anime style
• Minimalist
• Abstract

⚙️ System Prompts Power

Apa itu System Prompt?

Instruksi dasar yang diberikan ke AI sebelum interaksi dimulai. System prompt membentuk personality, behavior, dan constraint dari AI.

Expert Consultant

"Kamu adalah konsultan expert dengan 20+ tahun pengalaman. Selalu memberikan jawaban berbasis data, cite sources, dan offer solusi praktis."

Hasil: Jawaban lebih dalam, kredibel, dan actionable

Creative Writer

"Kamu adalah penulis kreatif pemenang award. Gunakan narrative techniques advanced, character development, dan storytelling yang engaging."

Hasil: Content lebih vivid, emotional, dan memorable

Ethical Advisor

"Kamu adalah advisor yang selalu mempertimbangkan aspek etika, keberlanjutan, dan dampak sosial dari setiap rekomendasi."

Hasil: Solutions yang lebih bertanggung jawab dan sustainable

🧪 Prompt Testing & Evaluation

📊 Framework Evaluasi

Relevance (Relevansi)
Clarity (Kejelasan)
Completeness (Kelengkapan)

❌ V1 (Poor)

Prompt: "Tulis artikel tentang AI"

Score: 2/10

Terlalu general, tidak ada konteks, format tidak jelas.

⚠️ V2 (Good)

Prompt: "Tulis artikel blog SEO-friendly tentang AI untuk marketer"

Score: 7/10

Better, tapi masih kurang spesifik tentang format dan tone.

✅ V3 (Excellent)

Prompt:

"Tulis artikel blog 1500 kata tentang 'AI untuk Digital Marketers'. Format: Intro hook, 5 section utama, CTA. Tone: friendly profesional. Include examples praktis, internal links suggestions, dan keyword focus: 'AI marketing tools'."

Score: 9/10 - Spesifik, clear format, actionable output

🔍 Testing Checklist

  • ✓ Output sesuai dengan ekspektasi format
  • ✓ Tone dan style konsisten dengan brief
  • ✓ Tidak ada hallucination atau informasi salah
  • ✓ Response time reasonable
  • ✓ Bisa handle variasi input
  • ✓ Follow constraints dan limitations
  • ✓ Output dapat digunakan tanpa banyak edit
